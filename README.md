# custom-fed37
Custom minimal installation of fedora 37 with the looks of Win11


<hr>


## Prerequisites
- Fedora Everything ISO (can be found at the alternatives downloads) 
- Install minimal
- Install wget & unzip (dnf install -y wget unzip)
